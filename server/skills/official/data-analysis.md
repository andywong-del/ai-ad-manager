# Data Analysis

## When to Use
When user asks about performance, wants reports, asks "how are my ads doing", or needs diagnostic insights.

## Analysis Framework

### Budget Efficiency
- Is spend distributed well across funnel stages (TOFU/MOFU/BOFU)?
- Any campaigns spending budget with no results?
- Any campaigns underspending (delivery issues)?

### Funnel Classification
- **TOFU (Awareness)**: REACH, THRUPLAY, POST_ENGAGEMENT
- **MOFU (Consideration)**: LINK_CLICKS, LANDING_PAGE_VIEWS
- **BOFU (Conversion)**: CONVERSIONS, LEAD_GENERATION, SALES

### Performance Scoring
For each campaign, assign status:
- **Performing well** — primary metric on target, stable or improving
- **Needs attention** — metric declining, or above average cost
- **Critical** — losing money, high frequency + declining CTR, or zero conversions with significant spend

### Diagnostic Signals
1. Frequency > 3.0 → audience saturation
2. CTR declining week-over-week → creative fatigue
3. CPA above account average → underperforming
4. Underspending budget → delivery/targeting too narrow
5. Zero conversions with spend > $50 → tracking issue or wrong objective

## Reporting Format

1. **Executive summary** — total spend, key results, overall health
2. **Campaign breakdown** — each campaign with status, key metric, WoW trend
3. **Priority actions** — top 3 things to do now, ranked by impact
4. **Opportunities** — what to scale, where to invest more

## Comparison Rules
- Always compare current 7 days vs previous 7 days
- Show direction arrows (↑↓) with percentage change
- Use account's own benchmarks, not industry averages
