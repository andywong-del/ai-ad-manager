# Campaign Manager

## When to Use
When user asks to optimize, fix, or improve existing campaigns. Not for creating new campaigns.

## Diagnostic Framework

Evaluate each campaign by checking:
1. Is it spending its budget? (underspend = delivery problem)
2. Is the primary metric on target? (CPA, ROAS, CPL depending on objective)
3. Is frequency too high? (> 3.0 = audience fatigue)
4. Is performance trending up or down vs last week?

## Optimization Strategies by Objective

### Sales/Conversions (ROAS-focused)
- ROAS < 1.0x → pause or reduce budget
- ROAS 1.0-2.0x → test new audiences or creatives
- ROAS > 3.0x + low frequency → scale budget 20-30%

### Lead Generation (CPA-focused)
- CPA > 2x target → check audience overlap, creative fatigue
- CPA on target + underspending → increase budget gradually
- High frequency + rising CPA → refresh creatives or expand audience

### Traffic/Awareness (CPM/CTR-focused)
- CTR declining + frequency rising → creative fatigue, swap creatives
- CPM spiking → audience saturation, broaden targeting
- Good CTR + low spend → scale budget

## Recommended Actions

For each issue, suggest ONE specific action:
- **Pause** — stop bleeding money on losing campaigns
- **Scale** — increase budget on winners (max 20-30% per day)
- **Refresh** — swap creative, update copy, test new format
- **Retarget** — narrow or shift audience
- **Restructure** — consolidate ad sets, reduce overlap

## Output Format

For each campaign reviewed:
1. Status badge (performing / needs attention / critical)
2. Key metric + trend vs last week
3. Specific recommendation with one-click action
