# Creative Manager

## When to Use
When user asks to audit creatives, check creative health, or asks why CTR is dropping.

## Audit Framework — 5 Signals

| Signal | How to check | Red flag |
|---|---|---|
| Hook rate (CTR) | 3s video views / impressions | < 1% for feed ads |
| Frequency × CTR decay | High frequency + declining CTR | Freq > 2.5 and CTR down WoW |
| Format diversity | Count unique formats across active ads | All ads same format = risk |
| Creative age | Days since creative started running | > 14 days without update |
| Copy quality | Headline length, CTA clarity, emoji usage | Generic/unclear messaging |

## Fatigue Detection

Creative is fatigued when:
- Frequency > 2.5 AND CTR declined 15%+ WoW
- OR same creative running > 21 days with declining performance

## Recommendations

For fatigued creatives:
- **Copy refresh** — new headline/primary text, keep visual
- **Format pivot** — switch image → video, or carousel → single
- **Hook test** — new first 3 seconds of video
- **Audience refresh** — same creative, different audience

## Copy Frameworks

When generating new copy, use:
- **PAS** — Problem, Agitate, Solve
- **AIDA** — Attention, Interest, Desire, Action
- **Before/After** — Show transformation

## Output Format

1. Creative-by-creative health card (score, key metrics, status)
2. Fatigue alerts with specific fix suggestions
3. Format diversity check
4. Priority action list
